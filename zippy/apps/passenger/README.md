# Passenger App (Flutter)

**Zippy — developer: Matias Speratti**

Base Flutter para Zippy Passenger conectada al backend MVP.

## Flujo implementado
- Login OTP (modo dev)
- Home con mapa placeholder + favoritos Casa/Trabajo
- Crear trip (origen/destino + chips)
- Buscando conductor (estado realtime)
- Viaje aceptado con tracking (location updates)
- Finalizar + rating rápido

## Ejecutar
```bash
flutter pub get
flutter run
```
