# Zippy Monorepo

**Zippy — developer: Matias Speratti**

Estructura inicial del MVP de Zippy para Firmat, Santa Fe, Argentina.

- `apps/passenger`: app Flutter para pasajeros.
- `apps/driver`: app Flutter para conductores.
- `apps/admin`: panel web placeholder.
- `services/backend`: API y tiempo real con NestJS.
- `docs`: documentación funcional y técnica (Docsify).
- `infra`: configuración local (Docker Compose y variables de entorno).
